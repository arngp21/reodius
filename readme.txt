ゲームタイトル : SpaceShooter（仮）

ジャンル : シューティングゲーム

特徴・アピールポイント : 弾の種類と敵の種類の豊富さによる高い難易度。
　　　　　　　　　　　　 「さまざまな弾やスキルを駆使してどうクリアするか」というやりこみ度。
　　　　　　　　　　　　 長いステージに加え、最後にはボスを配置。
　　　　　　　　　　　　 option
　　　　　　　　　　　　 画面に動きをつけるために、回転行列を使って敵が死んだときのエフェクトを作成。

制作ソフト : visualstadio2022

操作 : 十字キーで移動、１で弾発射、２でスキル使用

工夫した点 : 敵や弾の動きが種類ごとに違うので、std::fanctionでラッパーした関数ポインタで管理。
　　　　　　 レーザー弾が横長のBOXで当たり判定をとっているので、一度当たった敵にもう一度当たらないように。（レーザー弾にIDを持たせた）
            　privateやenumclass、Get関数などを使ったカプセル化。
　　　　　　 CseneManagerを使ったシーンの管理。
　　　　　　 Windowの中心を見るCameraとPlayerを一緒に動かし、横スクロールを実装。
　　　　　　 
              

開発期間 : 約３か月
