ゲームタイトル : reodius

ジャンル : シューティングゲーム

特徴・アピールポイント : 弾の種類と敵の種類の豊富さによる高い難易度。
　　　　　　　　　　　　 「さまざまな弾やスキルを駆使してどうクリアするか」というやりこみ度。
　　　　　　　　　　　　 長いステージに加え、最後にはボスを配置。
　　　　　　　　　　　　 画面に動きをつけるために、回転行列を使って敵が死んだときのエフェクトを作成。

制作ソフト : visualstadio2022

操作 : 十字キーで移動、１で弾発射、２でスキル使用

工夫した点 : 敵の動きが種類ごとに違うので、関数ポインタで管理。
　　　　　　 レーザー弾が一度当たった敵にもう一度当たらないようにした。
            　privateを使ったカプセル化。
              Get関数を使ったアロー演算子の又貸し防止。
              

開発期間 : 約３か月
